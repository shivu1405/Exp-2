# Exp - 2 LIBRARY MANAGEMENT SYSTEM

# AIM:
To design and develop a Library Management System that efficiently manages book records, member details, and borrowing transactions.

# SRS (Procedure):
1. Identify user requirements and system objectives.
2. Design the database for books, members, and transactions.
3. Create interfaces for admin and users.
4. Implement modules for issuing, returning, and searching books.
5. Integrate fine calculation for late returns.
6. Test all modules for correctness and efficiency.
7. Deploy the system for real-time use.
8. Maintain and update the system as needed.
